# Translations

MinDraft UI strings live in `src/lib/i18n/locales/`:

| File | Language |
|------|----------|
| `en.ts` | English (default) |
| `es.ts` | Spanish |
| `no.ts` | Norwegian (Bokmål) |

## Adding or changing copy

1. Add the key to `en.ts` first (source of truth).
2. Mirror the same key structure in `es.ts` and `no.ts`.
3. Use it in code with `t("section.key")` from `$lib/i18n`.
4. In Svelte components, wrap dynamic labels in `$derived` or `$derived.by` and read `localeState.locale` when the whole block must re-render.

## Parameters

Messages may include placeholders: `"Block {index}/{total}"`.

```ts
t("statusbar.block", { index: 1, total: 5 });
```

## Persistence

The chosen locale is stored in `localStorage` under `mindraft-locale`. Default is **English**.

## Scope

- User-facing UI: translated via locale files.
- Code comments and this documentation: **English**.
- Markdown command names (`:w`, `:saveas`, …): unchanged across locales.
